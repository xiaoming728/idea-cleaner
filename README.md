# IDEA Cleaner

一个用于清理 IDEA 项目中 node_modules 和 target 文件夹的桌面工具。

## 功能特性

- 🔍 扫描指定目录下的所有 node_modules 和 target 文件夹
- 📊 显示每个文件夹的大小和位置
- 🗑️ 支持单个删除和批量删除
- 💻 跨平台支持 (Windows, macOS, Linux)
- 🌙 暗色主题界面

## 技术栈

- Electron
- Vue 3
- Element Plus
- Vite

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建应用
npm run build
```
